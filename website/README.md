# Mercury Website

Docusaurus 3 site for [Mercury Agent](https://github.com/cosmicstack-labs/mercury-agent).

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

Build output goes to `../docs/` for GitHub Pages deployment.

## Deploy

The site is deployed to `mercuryagent.sh` via GitHub Pages from the `docs/` directory on the `main` branch.
